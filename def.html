Web Languages and Frameworks

Term Definition
C# C# is an object-oriented language which runs in .net environment released by Microsoft. It's useful for various purpose, including desktop applications development, game development.
Go Go language is a programming language published by google. It's being more important nowadays, for backend development, game development and cloud programming purposes. It has very interesting language features.
JavaScript JavaScript is almost everywhere nowadays.When you open a website,it's JavaScript that controls the interactive logic. Not only it can be used for front end development, but also for backend development purposes via node.js environment. Its name looks analogous to Java in the first glance, but they are totally irrelevant.
Java Java is a classic object-oriented programming languages which runs in Java Vritual Machine(JVM). Due to Jvm, Java has a Strong advantage to run in different platforms. It can be used as backend development language, Android development language, etc.
PHP PHP is a script programming language used for backend development. It's pretty lightweight, and easy to deploy. Common in small web projects
Python python is a script programming language which is becoming more and more popular due to its simplicity of syntax. Common in machine learning, science research, backend development.
SQL "SQL is a domain-specific language used in programming and designed for managing data held in a relational database management system (RDBMS). It is particularly useful in handling structured data, i.e. data incorporating relations among entities and variables."
Swift "Swift is a powerful and intuitive programming language for iOS. Swift code is interactive and fun, the syntax is concise yet expressive, and Swift includes modern features developers love."
Ruby "A dynamic, open source programming language with a focus on simplicity and productivity. It has an elegant syntax that is natural to read and easy to write."
WordPress "WordPress is a free and open-source content management system (CMS) written in PHP and paired with a MySQL or MariaDB database with supported HTTPS. WordPress was originally created as a blog-publishing system but has evolved to support other web content types."
